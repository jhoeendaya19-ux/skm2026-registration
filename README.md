# Sorsogon Kasanggayahan Marathon 2026 - Vercel Deploy Package

This folder contains the files for the first online Vercel deployment.

## Pages

- `index.html` - public registration website
- `admin.html` - admin dashboard

## Backend Services

This site connects to:

- Supabase database
- Supabase Storage
- Supabase Auth
- Supabase Edge Function for Resend emails

## First Deployment Goal

Deploy to a temporary Vercel URL first.

Example:

`https://your-project-name.vercel.app`

After deployment, test:

- Public registration
- Payment proof upload
- Registration received email
- Admin login
- Payment approval
- Bib number assignment
- Approval email
